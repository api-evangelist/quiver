# Quiver Quantitative

Quiver Quantitative (https://www.quiverquant.com/) is an alternative-data platform that aggregates non-traditional financial datasets and exposes them through a single REST API. The service is targeted at quantitative researchers, fintech developers, and individual investors who want programmatic access to data signals that go beyond traditional fundamentals and price data. Plans start at $10/month.

## Quiver Quantitative API

The Quiver API (https://api.quiverquant.com/) provides REST access to alternative financial datasets covering politics, government, corporate activity, retail behavior, and market microstructure. Data is delivered in JSON and is available across both bulk historical endpoints and live, ticker-specific lookups. Endpoints are organized under versioned path prefixes such as `/beta` and `/v1`, and authenticated requests are made over HTTPS using a token issued from the Quiver dashboard.

### Datasets

- Congressional Trading (House and Senate)
- Insider Trading
- Politician Net Worth
- Government Contracts
- Corporate Lobbying
- Corporate Donors
- Corporate Patents
- Executive Compensation
- Institutional Holdings
- ETF Holdings
- Top Shareholders
- Quiver Newsfeed
- Off Exchange Activity
- App Ratings
